# chess-server

Spring Boot application for chess server functionalities.

## Tech Stack
- Java 17
- Spring Boot 3.5.5
- Selenium
- ChessLib
- PostgreSQL

## Prerequisites
- Java 17
- Maven
- PostgreSQL

## Setup & Run
1. Clone the repository.
2. Ensure PostgreSQL is running and configured.
3. Run the application:
   ```bash
   mvn spring-boot:run
   ```
